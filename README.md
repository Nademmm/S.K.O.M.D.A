# SKOMDA
> Website Virtual Museum Interaktif Berbasis 3D untuk Edukasi Budaya dan Tata Tertib Sekolah

## 📖 Deskripsi

SKOMDA merupakan website interaktif yang mengubah cara penyampaian informasi mengenai budaya dan tata tertib sekolah menjadi pengalaman eksplorasi virtual layaknya bermain game.

Alih-alih membaca halaman website yang panjang dan membosankan, pengguna akan memasuki sebuah museum virtual 3D menggunakan perspektif First Person (FPV). Di dalam museum, pengguna dapat berjalan, mengeksplorasi ruangan, dan berinteraksi dengan berbagai objek yang berisi informasi mengenai budaya maupun peraturan sekolah.

---

# 📌 Latar Belakang

Sebagian besar website sekolah masih menggunakan konsep penyampaian informasi berupa teks panjang dan scrolling halaman.

Permasalahan tersebut menyebabkan:

- Pengguna cepat bosan
- Informasi tidak dibaca sampai selesai
- Tata tertib sekolah kurang dipahami
- Budaya sekolah sulit dikenalkan kepada siswa baru

SKOMDA hadir sebagai solusi dengan konsep **Gamified Learning Website**.

---

# 🎯 Tujuan

- Meningkatkan minat pengguna dalam membaca informasi sekolah.
- Mengenalkan budaya sekolah secara lebih interaktif.
- Menyampaikan tata tertib melalui media virtual.
- Memberikan pengalaman belajar yang menyenangkan.
